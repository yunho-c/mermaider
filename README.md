# Mermaider

Bun-powered SvelteKit starter with `shadcn-svelte` and Tailwind CSS v4.

## Stack

- Bun 1.3
- SvelteKit 2
- Svelte 5
- Tailwind CSS 4
- shadcn-svelte 1

## Scripts

```sh
bun run dev
bun run build
bun run preview
bun run check
```

## Scaffold Notes

This project was created with the current `sv` CLI and then initialized with
`shadcn-svelte`.

Installed starter UI components:

- `button`
- `card`
- `badge`
